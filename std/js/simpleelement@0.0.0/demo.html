By default the content would be colored red, as defined by the "style" tag present in the shadow root. 
  * { color: red; }
This can be overridden by defining styles on the "::part(content)" psudeo-element 
  red-element.green::part(content) {
    color: green;
  }
Additionally, elements passed in can be styled as normal 
  .blue {
    color: blue;
  }
